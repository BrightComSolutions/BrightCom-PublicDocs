## Release Summary

This release of BRC Core is primarily a maintenance and compatibility update to align with the latest BRC Logistics and BRC Connect versions. No new features or customer-facing changes are included in this build.

## Changes Included

- Dependency version updates to support BRC Logistics 19.4
- Platform compatibility improvements
- Internal code quality enhancements

## Breaking Changes

No breaking changes in this release.

## Notes

For significant feature work in progress on the Core product, including major Price Book performance enhancements for high-volume scenarios, see the Week 14 tracking document. These features are currently in development and will be released in a future update.
