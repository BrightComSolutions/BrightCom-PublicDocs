## Detailed Feature Information

---

### Customer Anonymization Control (#24284)

Adds a boolean field on the Customer card allowing users to mark customers for inclusion in the anonymization list. This enhancement supports data privacy compliance by providing granular control over which customer records should be anonymized.

---

### Customs Handling via API (#20489)

Integrates customs handling functionality through external API services. This feature enables automated customs document processing and streamlines international shipping workflows.
