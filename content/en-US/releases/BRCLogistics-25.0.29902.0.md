## Detailed Feature Information

### Business Central v28 Updates (#25510)

Essential compatibility updates to ensure BRC Logistics functions correctly with Business Central v28. This proactive update addresses API changes, platform updates, and feature deprecations introduced in the v28 release. Customers planning to upgrade to Business Central v28 should install this version to ensure seamless operation.

**Key Changes:**
- Platform API compatibility updates
- Code modernization for v28 requirements
- Deprecated feature handling
- Testing and validation for v28 environments

### Deposco US WMS Integration - Konges Sløjd (#23425)

Updates to the Deposco warehouse management system integration to support Konges Sløjd's US operations. Enhancements improve data synchronization and warehouse operation workflows.

## Installation Notes

This hotfix release is recommended for all BRC Logistics customers, especially those planning to upgrade to Business Central v28. The update can be installed through the standard AppSource update process.
