## Detailed Feature Information

### Variant Selection Prompt on Sales Returns (#24182)

Introduces a variant selection prompt when processing sales return lines for items with variants. This feature ensures accurate return processing by requiring users to explicitly select the correct product variant, reducing errors and improving inventory accuracy.

#### Variant Prompt on Sales Return Line (#24224)

Implements the variant selection prompt functionality directly on sales return line entry. When adding a return line for an item with variants, users are prompted to select the specific variant being returned.
